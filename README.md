# Chat App

A real-time online chat application built with a browser front end and a Node.js server using WebSockets. Users can join a chat room, see who is online, send messages instantly, and view live updates without refreshing the page.

## Features

- Real-time messaging
- Join chat rooms
- Online user list
- Typing indicator
- Timestamps on messages
- Simple and responsive interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express
- WebSocket (`ws`)
